Features

JsonXmlConvert is a tool which can handle json and xml. It have the following features:

1. Do the accuracy check of json’s or xml’s format and can be formatted output and can customize the output style.
2. Parse json or xml and show the data in the form of a dictionary.
3. Support interconversion of the most Json and xml, but not support attribute tansformation of xml for the time being.
4. Support UTF-8 coded text, furthermore can not be garbled.
5. Format the result of display and to support the color mark.
