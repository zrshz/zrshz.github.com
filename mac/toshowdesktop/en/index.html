Features

ToShowDesktop is a simple tool to show the desktop with the following major features:

1. The easy way to quickly display desktop.
2. You can use Dock icon or Status Bar icon to do this.
3. Now it supports hide Dock icon that few application can do.
4. Allow automatically start at login.
5. Support for global shortcuts.
6. Supports more languages.
