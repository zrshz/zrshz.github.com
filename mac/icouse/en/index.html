Features

Icouse is a software used to generate the common images. It have the following features:

1. Generate mac app icon and other images.
2. Support for multiple formats of open or save.
3. Quick preview the effect.
4. Very simple setup, drag or right mouse click to load images.
5. It has complete running information, you can see the specific running status.
6. Perfect support import svg files.
